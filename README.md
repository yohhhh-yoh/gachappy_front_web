# gachappy_front_web
gachappy web version front

## 概要
複数メーカーに分散しているカプセルトイ（ガチャ）の新作情報を一元管理し、
発売日・商品情報を簡単に確認できるWebアプリです。

## 主な機能
- 本日発売商品の表示
- 発売スケジュール一覧
- 商品詳細表示
- 注目商品ランキング
- 商品検索（商品名・メーカー）
- お気に入り機能

## 使用技術
### フロントエンド
- HTML / CSS / JavaScript

### インフラ
- Cloudflare Workers
